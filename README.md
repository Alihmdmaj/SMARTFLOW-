# SmartFlow+ | AI-Powered Traffic Management 🚀
**Location:** Tangier, Morocco 🇲🇦

### 🌟 Project Vision
SmartFlow+ is an AI-powered traffic management system designed to reduce congestion, lower CO2 emissions, and optimize urban mobility using hybrid-powered infrastructure.

### 🛠️ Current Progress (Hackathon PoC)
This repository contains the functional prototype of the User Interface, featuring:
* **Live Map Integration:** Real-time tracking in Tangier.
* **Smart Routing:** AI-simulated paths that color-code traffic (Green/Orange/Red).
* **CO2 Analytics:** Real-time calculation of emissions prevented by taking optimized routes.
* **IoT Parking Logic:** Live simulation of parking sensor data and availability.
* **Solar Infrastructure:** Monitoring of hybrid-power intake for traffic signals.

### 🚦 How to use the Demo
1. Search for a destination in Tangier (e.g., "Marina").
2. Press **Enter** to see the AI calculate the best route.
3. Check the **Environmental Impact** on the left panel.
4. Use the **Nearby Parking** button to find available IoT-monitored spots.
