# SmartFlow+ | AI-Powered Traffic Management 🚀
**Location:** Tangier, Morocco 🇲🇦

### 🌟 Project Vision
SmartFlow+ is an AI-powered traffic management system designed to reduce congestion, lower CO2 emissions, and optimize urban mobility using hybrid-powered infrastructure.

### 🛠️ Current Progress (Hackathon PoC)
This repository contains the functional prototype of the User Interface, featuring:
* ** Map Integration:** Display tracking in Tangier.
* **Smart Routing:** Paths that color-code traffic (Green/Orange/Red).
* **CO2 Analytics:** Display of CO2 emissions prevented by taking optimized routes.
* **IoT Parking Logic:** Simulation of parking sensor data and availability.

### 🚦 How to use the Demo
1. Press **Fined Best Route** to see the AI calculate the best route.
2. Check the **Environmental Impact** on the left panel.
3. Use the **Nearby Parking** button to find available IoT-monitored spots.
